## A2: 슬라이스 유틸리티

**디렉토리**: `a2-slice-ops/`

**내용**: `[]int` 슬라이스를 처리하는 유틸리티 함수들을 구현합니다.
- `Filter`: 조건에 맞는 요소만 추출
- `Map`: 모든 요소에 변환 함수 적용
- `Reduce`: 슬라이스를 단일 값으로 축약
- `Contains`: 특정 값 포함 여부 확인
- `Unique`: 중복 제거

```bash
cd a2-slice-ops
# sliceutil.go 파일의 함수 본문을 채우세요.
go test -v
```

**배점**: 20개 테스트, 100점 만점
